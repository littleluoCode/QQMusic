# QQMusic

#### 介绍
本项目为 QQ音乐PC端网页静态页面，包含大部分子页面，以及部分特效

博客：[littleluoの小站](https://littleluo.cn)

#### 项目结构
- QQMusic
  - artists           开放平台页面
  - client             客户端页面
  - css                 CSS 文件
  - font               字体图标文件
  - images          图片
  - js                    JS 文件
  - favicon.ico    标题图标文件
  - index.html    主页

#### 技术栈

- HTML5  CSS3  Javascript
- Git
- Scss / Sass
